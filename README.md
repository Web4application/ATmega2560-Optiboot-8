# ATmega2560 Optiboot 8.0 Repo

This repo contains the hardware definition and bootloader for Optiboot 8.0 on the ATmega2560.

## Why Optiboot 8.0?
- **Small Size:** Only 512 bytes.
- **Fast:** Uploads at 115200 baud.
- **Reliable:** Better support for programs > 128KB.

## Installation
1. Move the `avr` folder to `Documents/Arduino/hardware/optiboot/`.
2. Restart Arduino IDE.
3. Select `[Optiboot 8.0] Arduino Mega 2560`.

For Source code see http://github.com/Optiboot/optiboot
